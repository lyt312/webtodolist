## ToDoList是一个个人待办事项管理项目.

        TodoList可以实现以下功能：

        1.用户登陆才能使用待办清单系统，具备用户注销功能
        2.添加待办事项
        3.标记任务完成
        4.新建清单功能
        5.隐藏/显示已完成事项功能
        6.更改显示样式功能
        7.本地保存清单功能（下次登陆网站后还能看到以前的记录）
        8.查找代办事项功能
        9.排序功能

### 2、ToDoList使用方法：

        1.注册登录

        2.新建清单
        3.给指定清单添加事项

        4.编辑事项

        5.指定清单修改名称

        6.更改事项完成状态
        7.隐藏/显示已完成事项
        8.各清单中事项排序方式改变（按时间升序/按时间降序）
        9.更改主题
        10.事项搜索

        11.注销
